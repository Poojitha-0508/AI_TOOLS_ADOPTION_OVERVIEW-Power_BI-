# AI Tools Usage & Productivity Insights Dashboard

## 📊 Project Overview

This project presents an interactive **Power BI dashboard** that analyzes the usage, productivity impact, and adoption trends of AI tools from **2021 to 2025**.
The dashboard helps explore how different AI tools are used across industries, their productivity benefits, and regional adoption patterns.

## 🎯 Objectives

* Analyze **AI tool adoption trends over time**
* Compare **daily usage across popular AI tools**
* Understand **industry-wise AI tool utilization**
* Examine the relationship between **productivity gains and user satisfaction**
* Identify **regional adoption patterns**

## 📈 Dashboard Features

* **KPI Indicators**

  * Total AI Tools Analyzed
  * Total Users
  * Average Daily Usage
  * Productivity Gain
  * Satisfaction Level

* **Visual Insights**

  * Top AI Tools by Daily Usage
  * AI Tool Usage Across Industries
  * AI Tool Adoption Trend (2021–2025)
  * Productivity vs Satisfaction Analysis
  * Regional AI Tool Adoption

* **Interactive Filters**

  * Year
  * Industry
  * Purpose
  * Region

## 🛠 Tools & Technologies

* **Power BI** – Dashboard design and visualization
* **Power Query** – Data cleaning and transformation
* **DAX (Data Analysis Expressions)** – Calculations and measures

## 📂 Dataset

The dataset includes information about:

* AI tool usage statistics
* User counts
* Productivity impact
* Satisfaction scores
* Industry and regional distribution

*Note: The dataset is used for educational and analytical purposes.*

## 💡 Key Insights

* AI tool adoption increased significantly during the observed period.
* Productivity gains vary across industries and use cases.
* Some tools show higher satisfaction despite moderate usage levels.
* Regional adoption patterns highlight growing AI usage globally.

## 📌 Project Purpose

This project demonstrates **data analysis, visualization, and dashboard storytelling skills using Power BI**. It is intended for learning and portfolio purposes.
